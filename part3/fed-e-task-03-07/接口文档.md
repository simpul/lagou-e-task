# 接口说明



## 接口地址

https://shop.fed.lagounews.com/



## 状态码

- 200：成功

- 400：客户端错误，比如表单数据提交错误

- 410000：登录过期



## 接口权限



需要身份认证的接口需要在请求头中提供用户身份信息。



```
Authorization: Bearer Token数据
```



**注意：Bearer和 Token数据之间有一个空格。**
